Official Membership Register

Ruby New Zealand Incorporated

This is an overview of Ruby New Zealand’s official membership register. If you’d like to sign up, please fill out the form. Membership isn’t very fancy, but it does allow you to have your say about the committee and other society matters at the AGM held at Rails Camp each year.

You can check whether you’re a current member on the list below (you’ll need to verify your personhood first). If your details have changed or you’d like to be removed from the membership register please email secretary@ruby.nz, there’s currently no facility to update these yourself.

If you'd like your name removed from this page, even if you want to remain a member, please email secretary@ruby.nz.
